# Ice Spirit (Ледяной Дух)

#boss #snowy

Шестой босс. Ледяной дух в замороженной деревне.

## Характеристики
- **Локация:** [[02-zones/Snowy|Snowy]] (замороженная [[02-zones/Village|Village]])
- **Кристаллы:** 20-35
- **Абилки:** Frost Wave, Blizzard, Ice Shards

## Способности
1. **Frost Wave** (каждые 5с) — линейная AoE, замедляет 2с
2. **Blizzard** (каждые 8с) — AoE вокруг босса 120px
3. **Summon Ice Shards** (каждые 12с) — 3 миньона (80 HP, быстрые, исчезают через 10с)

## Дроп
- **Warmth Core** — гарантированный дроп
  - Legendary accessory: { hp: 60, dmg: 25, speed: 12, regenPercent: 3 }
  - Используется у campfire для восстановления деревни

## Код
- `VillageZone.js` (frozen режим)
- Спавн после убийства всех 32 зимних мобов
