# Shadow — Теневой Район

#zone #shadow

> Параллельное измерение, где тени живут своей жизнью.
> Здесь отражения реальности искажены.

## Общая информация

| Параметр | Значение |
|----------|----------|
| **Тип** | Измерение |
| **Порядок** | Зона 3 (после Cursed Lands) |
| **Босс** | Shadow King |
| **Тематика** | Тени, искажения, порталы |

## Враги

| Враг | Тип | Особенности |

| Параметр | Значение |
