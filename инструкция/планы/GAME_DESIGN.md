# GAME DESIGN

## Базовые характеристики игрока

### Уровень
- Максимальный уровень: 100
- Формула опыта для следующего уровня: `EXP_required = 100 * level^1.5`
- Каждый уровень: +10 HP, +2 к каждому атрибуту

### Атрибуты

| Атрибут | Влияние |
|---------|---------|
| Сила (STR) | +1.5 урон/очко |
| Ловкость (DEX) | +0.8% шанс крита/очко, +0.5% уклонения/очко |
| Интеллект (INT) | +2 урон заклинаний/очко, +5 MP/очко |
| Выносливость (VIT) | +15 HP/очко |
| Удача (LCK) | +1% шанс редкого лута/очко, +0.5% крита/очко |

### Здоровье
- Базовое: 100 HP
- Восстановление: 1 HP/сек после боя (5 сек задержка)

### Мана
- Базовая: 50 MP
- Восстановление: 2 MP/сек

├── Обычные враги (3-5 типов)

### Формула шанса выпадения

```javascript
base_chance = rarity_base
modified_chance = base_chance * (1 + luck * 0.01) * difficulty_multiplier
```

## Система боссов

### Механики боссов

Каждый босс имеет 2-3 уникальные способности.

**Древний дракон (биом 1):**
1. Огненное дыхание — линейная AoE атака (3 сек предупреждение)
2. Полёт — неуязвимость на 5 сек, затем удар с неба
3. Призыв волков — 2 волка каждые 30 сек

### Формулы боссов

```javascript
boss_hp = base_hp * difficulty_multiplier * (1 + floor * 0.1)
boss_damage = base_damage * difficulty_multiplier * (1 + floor * 0.1)
boss_exp = base_exp * difficulty_multiplier * (1 + floor * 0.2)
```

## Процент критического удара

```javascript
crit_chance = 0.05 + dex * 0.008 + lck * 0.005
crit_damage = 1.5 + (weapon.crit_bonus || 0)
```

## Процент уклонения

```javascript
dodge_chance = 0.02 + dex * 0.005
dodge_chance = Math.min(dodge_chance, 0.3)  // макс 30%
```

## Формула урона

```javascript
base_damage = weapon.damage + str * 1.5
actual_damage = base_damage * (1 - target.defense / (target.defense + 100))
crit_damage = actual_damage * crit_damage_multiplier
final_damage = crit ? crit_damage : actual_damage
```
