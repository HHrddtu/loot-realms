# Все текстуры

#textures #index

Процедурные текстуры (Canvas 2D), без внешних ассетов.

## Модули
| Файл | Содержит |
|------|----------|
| `zones.js` | meadow, forest, mine, cave, hell, village, snowy, cemetery, castle |
| `player.js` | hero, walk cycle, attack |
| `npcs.js` | merchant, blacksmith, elder, child_npc, sage, alchemist, angel |
| `items.js` | loot textures, equipment, potions, gems |
| `enemies.js` | goblin, skeleton, wolf, spider, bat, golem, wraith |
| `bosses.js` | treant, skeleton_lord, giant_bat, demon, ice_spirit |
| `expansion.js` | village textures |
| `effects.js` | projectiles: fireball, ice_shard, purify |
| `animations.js` | slash, hit, level_up, arrow, meteor |
| `snowy.js` | snow textures, frozen trees |
| `mine.js` | rocks, rails, crystals, chests |
