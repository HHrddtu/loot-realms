# IntroScene — Сцена интро

#scene #intro

## Описание
Короткая кат-сцена перед стартом игры. Показывается после выбора класса.

## Порядок экранов
```
1. Лого → "Loot Realms"
2. Выбор класса (ClassSelectScene)
3. Интро (3-5 секунд, текст + фон) ← ЭТА СЦЕНА
4. Village (GameScene)
```

## Интро по классам

| Класс | Текст |
|-------|-------|
| **Sage** | "Ты учился в библиотеке, когда тьма поглотила деревню. Твои знания — твоё оружие." |
| **Alchemist** | "Твои зелья спасали раненых. Теперь ты спасёшь всех." |
| **Angel** | "Ты упала с небес, защищая деревню. Теперь ты сражаешься за них." |

## Файлы
- `src/scenes/IntroScene.js` — сцена
- `src/config/intro.js` — тексты интро

## Время показа
3-5 секунд на каждый текст.
