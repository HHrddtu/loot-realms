# KeybindScene

#scene #ui #input

## Описание
Настройка клавиш управления. Доступна из Pause Menu → Advanced → Controls.

## Файлы
- `src/scenes/KeybindScene.js` — сцена
- `src/keybinds.js` — логика сохранения/загрузки

## Функции
- Просмотр текущих привязок клавиш
- Изменение привязок (клик → нажать клавишу)
- Сброс на дефолт
- Прокрутка списка

## Доступ
Pause → Advanced → Controls (или `P` → `K`)

## Клавиши по умолчанию
- `WASD` — движение
- `SPACE` — атака/взаимодействие
- `I` — инвентарь
- `T` — таланты
- `C` — крафт
- `Q` — квесты
- `F` — зелье
- `P` — пауза
- `TAB` — статы
