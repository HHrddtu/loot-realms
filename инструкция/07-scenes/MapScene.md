# MapScene — Экран карты

#scene #map

## Описание
Визуальная карта для навигации между зонами.

## Расположение зон на карте

```
┌─────────────────────────────────────┐
│  [Village] ──── [Depths]           │
│       │              │              │
│       ↓              ↓              │
│  [Forest]      [Cursed Lands]      │
│       │              │              │
│       ↓              ↓              │
│  [Castle]    [Shadow District]     │
│                      │              │
│                      ↓              │
│              [Tower] → [Throne]     │
└─────────────────────────────────────┘
```

## Элементы карты

| Элемент | Описание |
|---------|----------|
| Зоны | Кнопки (нажатие = переход) |
| Линии | Связи между зонами |
| Замки | Открыта/закрыта зона |
| Текст | Название зоны + описание |
| Кнопка "Назад" | Возврат в игру |

## Файлы
- `src/scenes/MapScene.js` — сцена
- `src/textures/zones.js` — текстуры зон

## Доступность зон
Зоны отображаются на карте только если они открыты (победа над боссом предыдущей зоны).
