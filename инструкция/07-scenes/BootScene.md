# BootScene — Загрузка

#scene #boot

## Функции
- Загрузка процедурных текстур (8s timeout)
- Проверка Firebase auth → Login или Menu
- Инициализация sound.js

## Ключевые методы
- `preload()`: генерация всех текстур
- `create()`: проверка auth, переход
