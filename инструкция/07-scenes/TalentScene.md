# TalentScene — Дерево талантов

#scene #talents

## Функции
- Дерево талантов со скроллом
- Class talents (47) + Account talents (70)
- 9 веток по 5-6 талантов
- Стоимость: 1/2/3/5/8 по row
- RESET кнопка (gold cost × level)
- LOCK кнопка (закрыть ветку)

## Размер
~330 строк
