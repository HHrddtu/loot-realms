# GAME VISION — Видение игры

#vision #design

**Loot Realms** — PvE Adventure Loot Game.

## Ключевое слово
**"Ещё один забег"** — каждый прогон должен ощущаться уникальным.

## Основной цикл
```
Исследование → Бой → Лут → Прогрессия → Новые сложности → Новые биомы
```

## Что чувствует игрок
1. **Рост силы** — каждый уровень и предмет делают заметно сильнее
2. **Награда за усилия** — редкие предметы ценны
3. **Достижение** — победа над боссом даёт гордость
4. **Интерес к новому** — каждый биом отличается
5. **Реиграбельность** — хочется пройти на higher сложности

## Философия
- Простота в освоении, глубина в механиках
- Каждый биом уникален
- Лут должен ощущаться ценным
- Сложность растёт постепенно

## Целевая аудитория
Игроки, любящие фарм, прогрессию и поиск редких предметов.
