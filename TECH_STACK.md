# TECH STACK

## Выбор технологии

### Движок/Библиотека: **Phaser 3**

| Критерий | Phaser 3 | PIXI.js | Three.js | Unity WebGL |
|----------|----------|---------|----------|-------------|
| Тип | Game Framework | Rendering | 3D Engine | Full Engine |
| 2D поддержка | Отличная | Отличная | Средняя | Хорошая |
| Физика | Встроенная | Нет | Встроенная | Встроенная |
| Размер | ~1MB | ~500KB | ~600KB | ~10MB+ |
| Простота | Высокая | Средняя | Низкая | Средняя |
| Сообщество | Большое | Среднее | Большое | Очень большое |
| Документация | Отличная | Хорошая | Хорошая | Отличная |

**Причина выбора:** Phaser 3 оптимизирован для 2D игр, имеет встроенную физику, collision detection, систему частиц и анимаций. Идеально подходит для 2D PvE-игры.

### Язык: **JavaScript (ES6+)**

| Критерий | JavaScript | TypeScript | C# |
|----------|------------|------------|-----|
| Распространённость | Очень высокая | Высокая | Высокая |
| Скорость разработки | Высокая | Средняя | Средняя |
| Безопасность типов | Нет | Да | Да |
| В browser | Нативно | Нужна компиляция | Нужен IL2CPP |

**Причина выбора:** Нативная поддержка браузером, максимальная скорость прототипирования. Для первого билда безопасность типов не критична.

### Формат данных: **JSON**

| Критерий | JSON | TOML | YAML |
|----------|------|------|------|
| Читаемость | Средняя | Высокая | Высокая |
| Размер | Маленький | Средний | Большой |
| Поддержка в JS | Нативная | Парсер | Парсер |
| Вложенность | Без ограничений | Ограниченная | Без ограничений |

**Причина выбора:** Нативная поддержка в JavaScript, минимальный размер, стандарт для веб-приложений.

### Хранилище: **LocalStorage + JSON**

| Критерий | LocalStorage | IndexedDB | Cookies |
|----------|--------------|-----------|---------|
| Объём | 5-10MB | Неограниченный | 4KB |
| Простота | Очень высокая | Средняя | Высокая |
| Синхронность | Да | Асинхронный | Да |
| Тип данных | Строки | Любой | Строки |

**Причина выбора:** Для первого билда LocalStorage достаточно. Простота использования, мгновенный доступ, синхронные операции.

### Рендеринг: **Canvas 2D (Phaser)**

| Критерий | Canvas 2D | WebGL | WebGPU |
|----------|-----------|-------|--------|
| Поддержка | 100% | 95% | 20% |
| Производительность | Средняя | Высокая | Очень высокая |
| Сложность | Низкая | Средняя | Высокая |

**Причина выбора:** Phaser 3 автоматически использует WebGL при возможности, с fallback на Canvas. Ручное управление не требуется.

### Система сборки: **Vite**

| Критерий | Vite | Webpack | Parcel |
|----------|------|---------|--------|
| Скорость | Очень высокая | Средняя | Высокая |
| Настройка | Минимальная | Сложная | Минимальная |
| HMR | Отличный | Хороший | Хороший |

**Причина выбора:** Максимальная скорость разработки, минимальная конфигурация.

## Ограничения

1. **Браузер:** Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
2. **Разрешение:** 1280x720 (базовое), масштабирование
3. **Производительность:** 60 FPS на среднем ПК
4. **Размер сборки:** < 5MB (без ассетов)
5. **Оффлайн:** Не требуется (первая версия)
