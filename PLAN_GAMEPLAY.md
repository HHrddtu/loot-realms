
### Что нужно

### Реализация

| `src/scenes/MapScene.js` | Новый файл: экран карты |

### Экран карты
```
┌─────────────────────────────────────┐
│  [Village] ──── [Depths]           │
│       │              │              │
│       ↓              ↓              │
│  [Forest]      [Cursed Lands]      │
│       │              │              │
│       ↓              ↓              │
│  [Castle]    [Shadow District]     │
│                      │              │
│                      ↓              │
│              [Tower] → [Throne]     │
└─────────────────────────────────────┘
```

### Элементы карты
| Элемент | Описание |
|---------|----------|
| Зоны | Кнопки (нажатие = переход) |
| Линии | Связи между зонами |
| Замки | Открыта/закрыта зона |
| Текст | Название зоны + описание |
| Кнопка "Назад" | Возврат в игру |

## 7. Бафы — мелкие и точечные

### Принцип
Каждый предмет даёт **1-2 маленьких бафа**, не "все сразу".

### Формат бафов

| Баф | Описание | Пример |
|-----|----------|--------|
| +3% урон | Маленький бонус | Оружие |
| +5% HP | Маленький бонус | Броня |
| +2% крит | Маленький бонус | Аксессуар |
| +1 реген/сек | Маленький бонус | Кольцо |
| +3% XP | Маленький бонус | Амулет |

### Не делать
- ❌ +50% ко всем
- ❌ +100% урон
- ❌ Все бафы на одном предмете

### Делать
- ✅ +3-5% на предмет
- ✅ 1-2 бафа на предмет
- ✅ Сеты дают суммарно +10-20%

## 11. Предыстория и лор

### Что нужно
Краткая предыстория для каждого класса + общий лор мира.

### Реализация
| Файл | Задача |
|------|--------|

### Лор мира

#### Общий
```
Двадцать лет назад герой спас деревню от Демона.
Деревня восстановилась и процветала.
Но под фундаментом росла тьма, забытая душами павших.
Теперь она пробуждается.
```

#### По классам

**Sage (Мудрец):**
```
Ты учился в библиотеке древнего храма, когда тьма поглотила деревню.
Твои знания — твоё оружие. Ты знаешь, что тьма не просто так.
Под деревнем спят души, которых не похоронили.
Ты должен найти их и дать покой.
```

**Alchemist (Алхимик):**
```
Твои зелья спасали раненых во время нападения Демона.
Ты знаешь, что тьма — это не просто магия.
Это химия души. Превращение. Трансмутация.
Ты сможешь найти противоядие, если найдёшь исходный материал.
```

**Angel (Ангел):**
```
Ты упала с небес, защищая деревню от Демона.
Твои крылья сломаны, но твоя сила не угасла.
Ты чувствуешь души被困ших под деревнем.
Ты должна освободить их и вернуть небесам.
```

#### Лор зон

**Depths (Подземелье Забвения):**
```
Под фундаментом деревни — древние катакомбы.
Здесь спят души всех, кого не похоронили за двадцать лет мира.
Они забыты, но не мертвы. Их тьма растёт.
```

**Cursed Lands (Проклятые Земли):**
```
На востоке от деревни — земли, проклятые древним злом.
Болота, ядовитые растения, мертвецы.
Это не просто проклятие — это корень зла, которое спало под деревней.
```

**Shadow District (Теневой Район):**
```
Параллельное измерение, где тени живут своей жизнью.
Здесь отражения реальности искажены.
Тени — это не просто тьма. Это забытые части нас самих.
```

**Tower of the Fallen King (Башня Павшего Короля):**
```
Двести лет назад Король правил этими землями.
Он был проклят и превращён в тень.
Его башня до сих пор хранит его воинов, верных присяге.
Освободи Короля — и он откроет путь к Трону.
```

**Throne of Eternity (Трон Вечности):**
```
Верх башни. Трон, пустой двести лет.
Здесь спит Повелитель Вечности — существо, поглотившего Короля.
Победи его — и цикл прервётся.
Мальчик из деревни станет новым Королём.
```
