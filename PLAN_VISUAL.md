## 1. Анимации атак

### Проблема
Сейчас атака — просто круг в центре. Не красиво, не понятно что происходит.

### Решение
Создать анимации ударов для каждого типа атаки.

### Типы атак

| Тип | Анимация | Описание |
|-----|----------|----------|
| **Удар мечом** | `attack_slash` | Дуга разреза, след клинка |
| **Выстрел** | `attack_arrow` | Летящая стрела с хвостом |
| **Заклинание** | `attack_spell` | Взрыв энергии, частицы |
| **Лечение** | `attack_heal` | Зелёные частицы вверх |
| **Удар щитом** | `attack_bash` | Волна отталкивания |

### Реализация

| Файл | Задача |
|------|--------|
| `src/textures/effects.js` | Добавить 5 текстур атак (каждая 32x32 или 48x48) |
| `src/textures/animations.js` | Добавить анимации для каждой (3-4 кадра) |
| `src/systems/Combat.js` | Заменить круг на анимацию по типу атаки |

### Текстуры атак

#### attack_slash (48x48)
```
Кадр 1: Начало дуги (тонкая линия)
Кадр 2: Середина (яркая дуга)
Кадр 3: Конец (след, угасание)
```

#### attack_arrow (32x8)
```
Кадр 1: Стрела летит (наконечник вперед)
Кадр 2: Стрела с хвостом
Кадр 3: Столкновение (вспышка)
```

#### attack_spell (40x40)
```
Кадр 1: Сбор энергии (сфера растёт)
Кадр 2: Взрыв (частицы разлетаются)
Кадр 3: Угасание (остатки)
```

#### attack_heal (32x32)
```
Кадр 1: Частицы вверх (зелёные)
Кадр 2: Яркая вспышка
Кадр 3: Рассеивание
```

#### attack_bash (48x24)
```
Кадр 1: Волна от щита
Кадр 2: Распространение
Кадр 3: Исчезновение
```

## 2. Проработка NPC моделей

### Проблема
NPC выглядят просто — minimum detail.

### Решение
Улучшить детализацию каждого NPC.

### NPC для улучшения

| NPC | Текстура | Что добавить |
|-----|----------|--------------|
| Старейшина | `villager_elder` | Седая борода, посох, морщины |
| Ребёнок | `child_npc_boy` | Больше деталей лица, прическа |
| Торговец | `villager_merchant` | Товары на поясе, сумки |
| Кузнец | `village_anvil` | Орнамент, свечение |
| Стражник | `forest_guard` | Доспехи, копьё, шлем |
| Охотник | `old_hunter` | Шкуры, лук, рюкзак |
| Жрица | `priestess` | Свечение, узоры, посох |
| Пленный рыцарь | `captured_knight` | Цепи, раны, сломанный меч |
| Павший ангел | `fallen_angel` | Сломанные крылья, аура |
| Пленная душа | `trapped_soul` | Полупрозрачность, цепи |
| Разведчик | `rift_scout` | Портальный посох, руны |
| Советник | `ghost_advisor` | Призрачность, свет |
| Принц | `trapped_prince` | Корона, цепи, рубашка |
| Мудрец | `tower_sage` | Книги, посох, мудрость |

### Пример улучшения (villager_elder)

**Было:**
- Голова, глаза, борода
- Роба
- Посох

**Стало:**
- Голова с морщинами
- Глаза с морщинами вокруг
- Борода с прядями
- Роба с узорами
- Посох с орнаментом
- Руки с перчатками
- Сапоги с застёжками

## 4. Красивое оформление аккаунт-эквипа

### Проблема
Аккаунт-эквип выглядит просто — нет визуального выделения.

### Решение
Добавить визуальные эффекты дляLegendary предметов.

### Что добавить

| Элемент | Описание |
|---------|----------|
| **Свечение** | Legendary предметы светятся |
| **Рамка** | Цветная рамка по rarity |
| **Анимация** | Пульсация для Legendary |
| **Иконка套装** | Визуал активного套装а |

### Реализация

| Файл | Задача |
|------|--------|
| `src/textures/effects.js` | Добавить glow-эффекты для rarity |
| `src/scenes/InventoryScene.js` | Добавить свечение для Legendary |
| `src/textures/ui.js` | Добавить рамки套装 |

### Визуал по rarity

| Rarity | Рамка | Свечение |
|--------|-------|----------|
| Common | Серая | Нет |
| Uncommon | Зелёная | Слабое |
| Rare | Синяя | Среднее |
| Epic | Фиолетовая | Яркое |
| Legendary | Золотая | Пульсирующее |
