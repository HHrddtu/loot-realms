## Текущие метрики (v0.16.0+)

| Метрика | Значение | Статус |
|---------|----------|--------|
| Bundle size | 2.3 MB | ⚠️ Требует code splitting |
| Textures generation | ~8s в BootScene | ⚠️ Блокирует загрузку |
| FPS (средний) | 60 (на среднем ПК) | ✅ |
| Particle systems | Создаются/удаляются без pool | ⚠️ |
| `recalcStats()` | Вызов при каждом изменении | ✅ Низкая нагрузка |
| `updateUI()` | Без throttle | ⚠️ |

## Проблемы производительности (из REVIEW.md)

1. **Бандл 2.3 MB без code splitting** — средняя серьёзность
2. **Все текстуры генерятся в BootScene** — блокирует загрузку, средняя серьёзность
3. **Particle системы без object pooling** — средняя серьёзность
4. **Нет throttle для некритичных обновлений** — низкая серьёзность
